Google Data


A day on the mountain, or especially an extended trip at a resort, requires resouces. Knowing where to find food, entertainment, supplies, and even medical attention is an important part of the planning process. The following categories for nearby business establishments were considered:

  • Restaurants
  • Bars
  • Spas
  • Shopping
  • Medical
  • Grocery
  • Lodging
The most important datapoints for each category were location and name. However, other datapoints such as rating and pricing would be ideal as well.


The [Google Cloud Platform] offers API access to a whole host of endpoints and services.

To gather information on businesses, the [Google Places API] was used.

An additional API service was used as well. There also exists a service which can provide coordinates and proper address information given a slightly less detailed name or address of an establishment. This is the [Google Geocoding API].


Aquiring the Data

There were two separate source scripts used in this process, and for a process which brought additional information to the ski resorts dataset.

  • See [Source - Google Places] for the Python code used to access the Places API.
  • See [Source - Google Functions] for the Python code used to create functions for the Places and Geocoding APIs.
